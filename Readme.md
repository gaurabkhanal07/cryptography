# Cryptography Tool in Python

A simple Python tool for encrypting and decrypting text. This project demonstrates basic cryptography concepts and provides a command-line interface to easily encrypt or decrypt messages. Perfect for learning purposes.

---

## 🚀 Features

- Encrypt and decrypt text
- Interactive command-line interface
- Option to repeatedly encrypt/decrypt in the same session
- Easy-to-understand Python code

---

## 🧾 Prerequisites

- Python 3.x
- Pip package manager

---

## 📦 Installation

1. clone the repository

```bash
git clone https://github.com/gaurabkhanal07/cryptography.git
cd cryptography

2.(Optional) Create a virtual environment
python -m venv .venv
source .venv/bin/activate   # Linux/macOS
.venv\Scripts\activate      # Windows

3. Install the requirements
pip install -r requirements.txt
